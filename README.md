# 项目简介

## 核心功能
- 基于YOLOv8的棉花幼苗生长阶段检测，支持5类识别：阔叶杂草、棉花第一阶段、棉花第二阶段、棉花第三阶段、杂草
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供自定义设置：可调整置信度阈值、IOU阈值，可选择显示标签和原图
- 支持多种模型选择，界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有检测结果，支持检测结果保存

## 适用场景
- 农业科研：棉花生长监测与分析
- 农田管理：杂草识别与早期干预
- 农业自动化：基于图像的棉花生长评估
- 教育研究：深度学习模型训练与应用研究

## 技术特点
- 基于深度学习框架，使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 技术栈包括：Python3.9、OpenCV、PyQt5、Ultralytics、Torch1.9
- 内置3000张标注数据集，包含5类棉花生长相关图像
- 提供完整源码、训练好的模型、数据集及运行教程
- 配置成功环境后可直接使用，支持图形化界面操作
